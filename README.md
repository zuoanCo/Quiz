# 刷题助手

面向本地题库和远程题库预留的 Electron 桌面刷题系统。首版采用 macOS 风格，主体验是专注练习台，同时包含题库管理、题目自动解析、AI 辅助解题、科目章节管理和远程题库设置。

## 已实现功能

- 单选、多选、判断、填空、简答、论述/分析六类题目。
- 科目管理：新增、编辑、删除。
- 章节管理：按科目新增、编辑、删除。
- 题库管理：新增、编辑、删除、筛选、搜索。
- 题目自动解析：粘贴原始题目文本，自动拆分题型、题干、选项、正确答案、解析，支持非选择题答案提取。
- AI 辅助解题：基于题干、答案和解析生成本地 AI 分析草稿，包括解题步骤、知识点、易错项和难度判断，后续可替换为远程模型接口。
- 随机练习：按全部题库或指定科目随机抽题，适合混合复习。
- 章节训练：按科目选择单个章节，展示章节题量、题型分布和题单进度，按章节题单推进练习。
- 提交后展示正确答案、解析和 AI 分析，不强制判定对错。
- 本地持久化：数据保存到本机 `localStorage`。
- 远程题库设置：API 地址、令牌、连接测试和接口结构预留。

## 开发启动

```bash
npm install
npm run dev
```

## 构建验证

```bash
npm run build
```

## 生成 Windows 安装包

```bash
npm run package:win
```

安装包输出到 `release/刷题助手-Setup-0.1.0-x64.exe`。

## 启动构建产物

```bash
npm run start
```

## 数据说明

首版数据默认写入浏览器本地存储；首次启动为空题库，不预置任何科目或题目。后续如果题库规模扩大，可按 `docs/TECHNICAL_DESIGN.md` 中的 Repository 设计替换为 SQLite 或远程同步。
